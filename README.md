# Subix

A movie lookup tool that help you search any film and instantly get its details, high-res posters, backdrops, subtitle links, and similar movie recommendations — all in one place.

## Features

- **Movie Search** — live autocomplete suggestions as you type, powered by the TMDB API
- **Movie Details** — poster, overview, release year, rating, vote count, and TMDB ID
- **Poster & Backdrop Gallery** — masonry grid of all available images with filter by type and language, shimmer skeleton loading animation, and lightbox viewer with keyboard navigation
- **4K Image Downloads** — download any poster or backdrop at original resolution directly from TMDB
- **Subtitle Links** — one-click redirect to SubDL for subtitle search
- **Download Link** — direct link to find the movie via CineSRC
- **Similar Movies** — browse related films and click to load them instantly
- **Hamburger Menu** — slide-in drawer with GitHub, Letterboxd links and an about section

## Tech Stack

- Vanilla HTML, CSS, JavaScript — no frameworks or build tools
- [TMDB API](https://www.themoviedb.org/documentation/api) for movie data and images
- Bebas Neue + DM Sans via Google Fonts

## File Structure

```
index.html    — markup and layout
style.css     — all styles and animations
app.js        — all logic and API calls
```

## Getting Started

1. Clone or download the repository
2. Open `index.html` in any modern browser
3. No installs, no build step, no API key setup needed — it works out of the box
